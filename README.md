# Aplicaci-n-para-control-de-muestras
Se realizo una aplicación mediante el uso de power apps que permite gestionar muestras de laboratorio, crear puntos de muestreo de manera dinámica y reportar resultados de manera dinámica mediante el uso de power apps.

# Entorno de Sharepoint

Se creo un entorno de sharepoint para centralizar las diferentes funciones.

<img width="799" height="521" alt="image" src="https://github.com/user-attachments/assets/f6a9de5f-cd2c-48fc-84a1-0b87c4440504" />

En esta se puede acceder a la aplicación general, la primera parte muestra una galeria con la lista de análisis con un código único para cada informe.

<img width="1352" height="522" alt="image" src="https://github.com/user-attachments/assets/694b265d-50bd-412a-a760-73df80fa4617" />

de aqui se puede acceder a una pantalla donde se configuran los diferentes paramétros y se guardan los resultados. 

<img width="1872" height="686" alt="image" src="https://github.com/user-attachments/assets/45491e17-c8a4-4153-a4b1-3991e07ddf1a" />

La tabla de análisis se crea de manera dinámica de acuerdo al cliente, sus puntos de muestreo y análisis desde un entorno que permita ingresar la información de los clientes. La tabla de paramétros permite seleccionar que limites de control se muestran y que limites no en el informe final.

Luego de esto se puede pasar a un entorno en el que se genera un reporte en formato pdf

<img width="796" height="135" alt="image" src="https://github.com/user-attachments/assets/44694afe-8149-4cb9-9dcb-ab24b028795c" />

En este se puede agregar gráficas con las tendencias de los valores históricos el informe final es compilado mediante un flujo de power automate generado con HTML.
